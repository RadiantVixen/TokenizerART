# Deploying Monkey42NFT on Remix

1. Open Remix IDE: [https://remix.ethereum.org](https://remix.ethereum.org)
2. Create a new file `Monkey42NFT.sol` and paste the contract code.
3. Compile the contract using Solidity version `^0.8.17`.
4. Go to **Deploy & Run Transactions**:
   - Environment: Injected Web3 (connect MetaMask)
   - Contract: `Monkey42NFT`
5. Click **Deploy** and confirm in MetaMask.
6. Copy your contract address and verify it on Etherscan.
